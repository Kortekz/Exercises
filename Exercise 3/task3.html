7. Pseudo-classes
:active: This pseudo-class selects an element when it is being activated by the user, typically when the mouse button is held down.

:first-child: Selects an element that is the first child of its parent.

:nth-child(): Selects elements based on their position among a group of siblings.

:not(): Allows you to select elements that do not match a given selector.

Pseudo-elements:
::before: Inserts content before the content of the selected element.

::after: Inserts content after the content of the selected element.

::first-line: Styles the first line of the selected element.

::selection: Styles the portion of an element that is selected by a user.
